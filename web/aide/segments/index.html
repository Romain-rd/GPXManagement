✂️ Segments

Découper en segments

Divisez une trace en portions nommées — un col, une boucle, une phase de montée — et comparez leurs statistiques. Les segments se surlignent sur la carte et le profil.

La section Segments apparaît pour les activités qui parcourent une distance (vélo, marche, rando, ski…) ; pas pour les séances sur place (muscu, escalade).

Créer des segments

  1. 1

    À la main, sur le profil

    Dans la section Segments (ou sur le profil), glissez horizontalement sur le graphe pour sélectionner une portion : un segment est créé sur cette plage.

  2. 2

    Automatiquement, via « Découper »

    Le menu Découper remplace les segments par un découpage régulier :

    Par distance
    Tous les 1 km · 5 km · 10 km
    Par temps
    Toutes les 30 min · heure · 2 heures
    Par phase
    Montées, descentes, pauses

Lire & gérer

  1. 3

    Le tableau des segments

    Chaque segment affiche Nom, Distance, D+, D−, Durée et Vitesse moy. Cliquez le nom pour le renommer ; l'icône carte ouvre le segment en plein écran ; l'icône corbeille le supprime. Tout supprimer efface le découpage.

  2. 4

    Surligner un segment

    Cliquez une ligne pour surligner ce segment sur la carte et le profil (re-cliquez pour désélectionner). Idéal pour isoler la montée d'un col et lire ses chiffres.
