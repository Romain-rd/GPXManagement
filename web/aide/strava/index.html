Strava

Importer vos activités Strava

Strava permet de télécharger l'archive complète de vos activités. Voici comment l'obtenir, puis l'importer dans GPXManagement.

L'export complet n'est disponible que sur le site web de Strava (pas dans l'app mobile), depuis un ordinateur. Strava prépare l'archive puis vous l'envoie par e-mail — cela peut prendre de quelques minutes à plusieurs heures.

Sur Strava — demander l'archive

  1. 1

    Connectez-vous à Strava sur le web

    Depuis un ordinateur, ouvrez strava.com et connectez-vous à votre compte.

  2. 2

    Ouvrez les Paramètres

    Cliquez sur votre photo de profil en haut à droite, puis sur Paramètres (Settings).

  3. 3

    Allez dans « Mon compte »

    Dans le menu de gauche, choisissez Mon compte (My Account).

  4. 4

    Section « Télécharger votre compte »

    Faites défiler la page Mon compte jusqu'à la section Télécharger votre compte (« Ces archives comprendront vos activités, photos, publications, itinéraires, etc. ») et cliquez sur Commencer.

  5. 5

    Demandez le téléchargement

    Sur la page Télécharger les données de votre compte, cliquez sur Demander le téléchargement. Strava confirme que le lien sera envoyé à votre adresse e-mail.

  6. 6

    Récupérez le fichier par e-mail

    La préparation peut prendre plusieurs heures. Quand l'e-mail de Strava arrive, ouvrez-le et téléchargez le fichier .zip de vos archives.

Dans GPXManagement — importer l'archive

  1. 7

    Ouvrez l'import Strava

    Dans la barre de menus, allez dans Fichier ▸ Importer un export Strava (ZIP ou dossier)…

  2. 8

    Sélectionnez le .zip téléchargé

    Choisissez le fichier export_…​.zip (ou le dossier déjà décompressé). GPXManagement extrait les traces, détecte le type d'activité et les ajoute à votre bibliothèque — les doublons sont ignorés automatiquement.
